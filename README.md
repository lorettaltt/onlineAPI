# onlineAPI
###使用说明

该平台采用`nodejs` + `mongodb`搭建，启动服务前请确保已安装好node及mongodb的环境

1. clone代码至本地

    ```
    git clone https://github.com/lorettaltt/onlineAPI.git onlineAPI
    ```

2. 安装依赖

    ```
    npm install
    ```

3. 启动服务

    ```
    node app.js
    ```

4. 访问

    ```
    http://localhost:1337
    ```
